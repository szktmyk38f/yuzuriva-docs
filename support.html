📱 ユズリバとは

ユズリバは、グッズを手軽に交換できるアプリです。推しグッズ、アニメグッズ、アイドルグッズなど、大切なコレクションを安心して譲渡・交換できます。

🚀 はじめ方

  1. アカウント作成 - メールアドレス、Apple ID、またはGoogleアカウントで登録
  2. プロフィール設定 - ユーザー名とプロフィール画像を設定（任意）
  3. 利用規約に同意 - 初回ログイン時に利用規約を確認
  4. 商品を出品または検索 - 交換したいグッズを探したり、自分のグッズを出品

❓ よくある質問（FAQ）

Q1. 会員登録は無料ですか？
はい、基本的な機能は無料でご利用いただけます。プレミアムプラン（有料）では、広告非表示や優先サポートなどの特典があります。
Q2. 商品を出品するにはどうすればいいですか？
ホーム画面の「+」ボタンをタップし、商品の写真を撮影またはアップロード、商品情報（タイトル、説明、カテゴリー、状態）を入力して「出品する」をタップします。
Q3. 取引の流れを教えてください
  1. 気になる商品を見つけたら、チャットで取引相手とやり取り
  2. 交換条件や配送方法を確認
  3. 双方が同意したら取引成立
  4. 商品を発送し、配送状況を共有
  5. 商品受け取り後、相互評価を行う
Q4. 配送方法はどうすればいいですか？
配送方法は取引相手と相談して決めてください。追跡可能な配送方法（らくらくメルカリ便、ゆうパック等）の利用を推奨します。
Q5. 手数料はかかりますか？
基本的な取引手数料は無料です。ただし、配送料は各自で負担していただく必要があります。
Q6. アカウントを削除したい
アプリ内の「設定」→「アカウント管理」→「アカウントを削除」から削除できます。削除すると、すべてのデータが完全に削除され、復元できませんのでご注意ください。
Q7. プッシュ通知をオフにしたい
アプリ内の「設定」→「通知設定」から、通知のオン/オフを切り替えられます。または、デバイスの設定からアプリの通知を無効化できます。
Q8. パスワードを忘れました
ログイン画面の「パスワードを忘れた方」をタップし、登録メールアドレスを入力すると、パスワードリセット用のリンクが送信されます。
Q9. 不適切な商品や迷惑なユーザーを見つけました
各商品ページやユーザープロフィールから「通報」機能をご利用ください。運営チームが内容を確認し、適切に対応いたします。
Q10. プレミアムプランの解約方法は？
iOS: 設定アプリ → Apple ID → サブスクリプション → ユズリバ → サブスクリプションをキャンセル
Android: Google Play ストア → メニュー → 定期購入 → ユズリバ → 定期購入を解約

⚠️ 注意事項

  • 金銭の直接取引は禁止されています
  • 偽物・違法コピー品の出品は禁止
  • 個人情報（住所、電話番号など）の公開は避けてください
  • 取引トラブルは当事者間で解決をお願いします（運営はサポート可能）
  • 商品の状態は正確に記載してください

🔒 プライバシーとセキュリティ

ユズリバは、ユーザーの皆様のプライバシーとセキュリティを重視しています。

  • すべての通信は暗号化されています
  • 個人情報は厳重に管理されます
  • 第三者への情報提供は、法令に基づく場合を除き行いません
